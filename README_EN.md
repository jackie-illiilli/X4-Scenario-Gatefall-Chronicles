# Gatefall Chronicles

One-line summary: Gatefall Chronicles is an original hidden Timelines campaign for X4, centered on a future Xenon crisis, anomalous jump technology, and the forced alliance of multiple factions against extinction.

The current public build is the Phase 1 demo of Mission 01: Under the Surface, and it is already fully playable from start to finish with subtitles, cutscenes, objectives, and scoring.

Chinese version: [README.md](README.md)

## Requirements

This mod currently requires:

* X4: Foundations
* X4: Timelines
* X4: Cradle of Humanity

## Installation

If you downloaded the project from GitHub, install it like this:

1. Download or clone this repository.
2. Copy the entire `extensions/gatefall_chronicles` folder into your X4 `extensions` directory.
3. Start the game and make sure `Gatefall Chronicles` is enabled in the Extensions menu.

Typical folder structure:

```text
X4 Foundations/
└─ extensions/
   └─ gatefall_chronicles/
```

## How to start the campaign

This is not a normal sandbox mission chain. It is implemented as a hidden Timelines campaign entry.

To access it:

1. Launch the game and enter `Timelines`.
2. Enter the Timelines Hub, the black void area between scenarios.
3. Find the hidden campaign / hidden scenario access point in the Hub.
4. Open the hidden scenario list.
5. Scroll to the end of that hidden scenario list.
6. Select `Gatefall Chronicles / Under the Surface`.

If you have already used the official hidden scenario terminal before, the flow is basically the same. Our campaign is attached to that same hidden scenario list.

## Current public content

### Campaign 01: Under the Surface

Current status: Public demo available and playable

Summary:

* The player and a Terran allied wingman enter the Mists of Artemis to deploy an advanced satellite.
* The mission escalates after the Kha'ak nest is found destroyed and an anomalous signal appears.
* The player is pulled through abnormal spatial folding into Savage Spur.
* After a breakout and outpost defense sequence, the Terran intervention battleship `TER Paradigm` arrives to take over the battlefield.

Current demo features:

* Full subtitle presentation
* Multi-stage mission objectives
* Intro, anomaly, warp, reinforcement, and finale cutscene sequences
* End-of-scenario scoring and star rating

Notes:

* The currently public content is the first mission demo
* No voice acting yet; subtitles only

## Notes

* The current public build is mainly meant to validate the first mission experience and the overall campaign direction.
* The campaign is still in active development. Later missions, presentation, scoring details, and localization will continue to improve.
* The published content currently uses a hidden Timelines campaign entry and does not replace the official Timelines campaign.

## Thanks

If you encounter scripting, cutscene, scoring, or access issues, feel free to open a GitHub issue.

Several missions in this campaign are also planned as tributes to classic games and campaigns. For example, A Faint Hope is intended as a tribute to the mission of the same name from Starsector, while Crime and Punishment is planned as a tribute to the well-known Mental Omega 3.3.6 campaign mission of the same name for Red Alert 2.
