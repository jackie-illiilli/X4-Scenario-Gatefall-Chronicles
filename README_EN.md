# Gatefall Chronicles

One-line summary: Gatefall Chronicles is an original hidden Timelines campaign for X4, built around anomalous Xenon jump technology, the collapse of the gate front, and a chain of desperate alliances made under extinction-level pressure.

The current public build already includes two playable demo missions:

- `Gatefall 01: Under the Surface`
- `Gatefall 02: Passing the Buck`

Chinese version: [README.md](README.md)

## Requirements

This mod currently requires:

- X4: Foundations
- X4: Timelines
- X4: Cradle of Humanity
- X4: Split Vendetta

## Installation

If you downloaded the project from GitHub, install it like this:

1. Download or clone this repository.
2. Copy the entire `extensions/gatefall_chronicles` folder into your X4 `extensions` directory.
3. Start the game and make sure `Gatefall Chronicles` is enabled in the Extensions menu.

Typical folder structure:

```text
X4 Foundations/
└─ extensions/
   └─ gatefall_chronicles/
```

## How to start the campaign

This is not a normal sandbox mission chain. It is implemented as a hidden Timelines campaign entry.

To access it:

1. Launch the game and enter `Timelines`.
2. Enter the Timelines Hub, the black void area between scenarios.
3. Find the hidden campaign / hidden scenario access point in the Hub.
4. Open the hidden scenario list.
5. Scroll to the end of that list.
6. Select `Gatefall Chronicles / Gatefall 01: Under the Surface`.

If you have already used the official hidden scenario terminal before, the flow is basically the same. Our campaign is attached to that same hidden scenario list.

## Current public content

### Campaign 01: Gatefall 01: Under the Surface

Current status: Public demo complete and fully playable with end-of-mission results

Summary:

- The player and a Terran allied wingman enter the Mists of Artemis to deploy a long-range reconnaissance satellite.
- The mission escalates after the Kha'ak nest is found destroyed and an anomalous signal appears.
- The player is dragged by abnormal spatial folding into the Xenon core region of Savage Spur.
- After a breakout and outpost defence sequence, the Terran intervention battleship `TER Paradigm` arrives to take over the battlefield.

Current demo features:

- Full subtitle presentation
- Multi-stage mission objective flow
- Cutscene sequences for the anomaly, warp transition, reinforcements, and finale
- End-of-scenario scoring and star rating

### Campaign 02: Gatefall 02: Passing the Buck

Current status: Public demo complete and fully playable with end-of-mission results

Summary:

- The player takes the role of Argon rear commander and remotely directs HAT's 86th Escort Fleet from Hatikvah's Choice.
- The mission escalates through overlapping Xenon attacks on a TEL freighter, a HAT miner, and a HAT station in Tharka's Cascade.
- After discovering that a Xenon capital concentration is forming deeper in the sector, the player orders the fleet leader to deploy a satellite and then hold the line.
- When Argon decides to sacrifice the 86th, the player steals a Yaki navigation protocol and diverts the Xenon offensive into Zyarth territory to buy the fleet a way out.

Current demo features:

- Station-interior start and dynamic interior progression
- Remote OOS fleet command through the map
- Multi-stage defence, recon, database infiltration, and navigation beacon deployment
- Full subtitles, BGM transitions, scoring, and star rating

Notes:

- The current public build is still subtitle-only; there is no voice acting yet.
- The first two missions can already be played through independently from start to finish.

## Notes

- The current public build is mainly meant to validate the first two missions, the narrative direction, and the campaign's underlying systems.
- The campaign is still in active development. Later missions, presentation, scoring details, and localization will continue to improve.
- The published content currently uses a hidden Timelines campaign entry and does not replace the official Timelines campaign.

## Thanks

Thanks to the X4 community, the official game and DLC mission files that made technical research possible, and everyone willing to test this hidden campaign demo.

If you encounter scripting, cutscene, scoring, or access issues, feel free to open a GitHub issue.

Several missions in this campaign are also planned as tributes to classic games and classic campaign design. For example, `A Faint Hope` is intended as a tribute to the mission of the same name from Starsector, while `Crime and Punishment` is planned as a tribute to the well-known Mental Omega 3.3.6 campaign mission of the same name for Red Alert 2.
