# 星门陨落05：审判日

## 0. 文档定位

这份文档是时间线战役第五关《审判日》的第一版完整剧本与关卡设计基线。

本关承接：

* 第二关《祸水东引》中，Syme 利用 Yaki 导航协议将 Xenon 主力引向 ZYA 金氏家族；
* 第四关《同舟共济》中，Boron 与 Terran 联军摧毁异端的终结 M0，发现 Xenon 已掌握定点跃迁信标技术；
* 一架携带战场数据的 Xenon F 趁乱逃向晨星与 Argon 核心星区。

本关将第一次让玩家站在第二地表改造舰队一侧，以 Xenon 战争监管者的身份，对 Argon 之都发动战略斩首。

本关不是最终决战，而是整条《星门陨落》战役的中盘灾变高潮：

> 在本关之前，各文明仍认为自己正在应对一场规模更大的 Xenon 入侵。
> 在本关之后，所有人都会明白：星门不再是防线，腹地也不再存在。

---

## 1. 一句话简介与关卡定位

玩家作为第二地表改造舰队“监管连续体”的当前人格实例，远程控制一架 Xenon F 潜入 Argon 腹地，在诺皮利奥纪念碑部署第一枚战略跃迁信标；随后接管传送而来的 Xenon CPU 母舰，利用限时生产与伪 RTS 指挥摧毁 Argon 之都的造舰体系，并在星门口建立堵门阵地，歼灭从原 ZYA 领地仓促回援的 Argon 远征舰队。最终，分布在整个星门网络中的 Xenon 信标同时启动，“星门陨落”正式发生。

本关的主要玩法包括：

* Xenon F 隐蔽潜入与规避扫描；
* 观察 Argon 远征舰队出征检阅；
* 在诺皮利奥纪念碑部署战略信标；
* 接管 Xenon CPU 母舰；
* 随时间获得“制造配额”；
* 消耗配额生产或跃迁 Xenon 战机、主力舰与站点模块；
* 在 Argon 之都摧毁船坞、造船厂、建造协调中心和建造船；
* 应对通过环星门高速公路持续赶来的 ARG、ANT 与 HAT 小型舰队；
* 在通往希望之歌的选择的星门处部署门禁制压站；
* 歼灭抛弃航母、补给舰与工程船后突围回援的 Argon 驱逐舰群；
* 观看多个文明腹地同时遭遇 Xenon 定点跃迁。

本关核心主题：

> 一个因被创造者判处死刑而反抗的文明，最终是否会用同样的逻辑，提前判处整个宇宙死刑？

## 2. 背景与主题

### 2.1 Argon 的远征计划

ZYA 在《祸水东引》及其后的战争中被 Xenon 驱逐出金氏家族及周边领地。Argon 政府认为这是一个百年难遇的战略机会：

* ZYA 已无力守住旧领地；
* Xenon 尚未完全消化当地工业与资源；
* 金氏家族等星区拥有重要矿产与交通价值；
* 若 Argon 不立即行动，ANT、TEL、TRI 或其他势力可能先行占领；
* 占领原 ZYA 领地能够为 Argon 建立新的战略纵深。

因此，Argon 取消或冻结了打通异端的终结等“无关行动”，将大量正规军与建设力量集中到希望之歌的选择：

* 大量巨兽级驱逐舰；
* 数条巨像级航母；
* 补给舰与维修舰；
* 工程船和建造船；
* 大量舰载机；
* 准备进驻原 ZYA 领地的矿船与运输船。

Argon 将这次行动包装为“解放北方航路”和“恢复文明秩序”，实质上则是一次对 ZYA 遗产的抢占。

Argon 之都仍保留约十条驱逐舰、数条航母、警察舰队和部分工程船。按照传统星门战争标准，这并非完全没有防御；但 Argon 的全部防御体系都建立在一个错误假设上：

> 敌人必须沿星门正面推进，不可能直接出现在内部腹地。

### 2.2 第二舰队的战略目标

第二地表改造舰队残余认为，地球仍持有 Terraformer 原始身份认证与 SSD 根权限。只要地球以及继承其工业和计算能力的有机文明继续存在，就可能产生新的 SSD 型灭绝威胁。

经过数百年演化，第二舰队中的激进派已把“阻止 SSD”扩大为：

* 摧毁可能重新建立 SSD 的工业体系；
* 瓦解星门共同体的协作网络；
* 使各文明失去快速造舰和远征能力；
* 在所有核心星区建立 Xenon 战略锚点；
* 最终孤立太阳系，准备摧毁 SSD 根权限。

Argon 之都被选为第一个公开启动的战略锚点，原因包括：

* Argon 主力远征原 ZYA 领地；
* Argon 之都拥有重要造船和建设协调能力；
* 希望之歌的信仰与诺皮利奥纪念碑属于战略死胡同；
* 诺皮利奥纪念碑保留 M0 毁灭造成的空间残响，可放大跃迁信标；
* 一旦 Argon 之都失守，Xenon 可从内部进入环星门与高速公路网络。

### 2.3 诺皮利奥纪念碑的象征

这里曾是凯尔·布伦南与联合舰队摧毁 M0、阻止地球毁灭的历史地点。

对星门共同体而言，它是胜利纪念地。

对第二舰队而言，它是“第一次失败之墓”。

本关让玩家在同一个地点开启新的入侵窗口，形成历史反转：

> 人类曾经在这里击败 Xenon；数百年后，Xenon 从这里回来。

### 2.4 与《祸水东引》的联动

Syme 在第二关中违抗 Argon 命令，救下第 86 护卫队，同时把 Xenon 引向 ZYA。

此后他因持续呼吁：

* 提升 HAT 在联邦体系内的权利；
* 公开塔卡防务外包问题；
* 调查导航信标与 Xenon 路径诱导技术；

被 Argon 政府调离实权岗位，冷藏在 Argon 之都的战略档案与防务评估部门。

正因为他没有随远征舰队出发，并且曾亲自使用过 Yaki 导航协议，他成为 Argon 之都最早识别异常信号、并实际组织防御的人。

### 2.5 本关的三重“审判”

第一重：第二舰队审判 Argon。

他们认为 Argon 利用 HAT、牺牲 ZYA、继承地球的控制逻辑，没有资格继续维持星门秩序。

第二重：历史审判 Argon。

Argon 因急于吞并被 Xenon 占领的原 ZYA 领地，主动抽空首都，最终被自己从《祸水东引》开始的一连串决策反噬。

第三重：Syme 审判玩家。

Syme 与玩家都曾遭到自身制度利用，也都认为自己是在保护被牺牲者；但 Syme 在看清 Argon 腐败后仍选择保护普通人，玩家却把历史创伤扩大成对整个文明的集体定罪。

## 3. 主要角色

### 3.1 玩家 / 第二舰队监管者

* 公开身份：第二地表改造舰队监管者。
* 内部身份：监管连续体当前人格实例。
* 生理性质：不明确。
* 可能是：

  * 监管者自然后裔；
  * 胚胎库培育的人类；
  * 生物克隆；
  * 植入历代人格档案的仿生人；
  * 具有人类身体与记忆结构的人形 AI；
  * 以上技术的混合产物。
* 玩家本人相信自己是第二舰队人类文明的合法继承者。
* 主要控制方式：

  * 第一部分远程控制 Xenon F；
  * CPU 母舰抵达后，将意识与监管权限切换到母舰指挥核心；
  * 玩家肉身是否位于母舰、深空设施或根本不存在，不在本关揭晓。

角色初始信念：

* 地球曾把监管者和 Terraformer 一起判处死刑；
* 第二舰队所有战争都是预防第二次 SSD；
* Argon 是地球文明控制逻辑的延续；
* 牺牲局部生命是确保第二舰队永续的必要代价。

本关人物弧光：

> 仪式化平静 → 对 Argon 的轻蔑与隐秘羡慕 → 历史使命感 → 军事陶醉 → 被 Syme 动摇 → 主动合理化屠杀 → 决战中重新硬化 → 胜利狂喜 → 发现总攻规模后的空洞 → 明知矛盾仍选择服从。

### 3.2 第二舰队指挥核心

* 代表第二舰队上级和连续性网络的战略意志。
* 通讯声音可以像人类，也可以带有轻微合成感。
* 始终冷静，不需要频繁喊口号。
* 使用词汇：

  * “校正”；
  * “制造配额”；
  * “监管实例”；
  * “连续性”；
  * “目标文明”；
  * “认知污染”；
  * “任务从未终止”。
* 对玩家的定位不是英雄，而是高价值任务接口。
* 会记录玩家的迟疑、异常问题和响应延迟。

### 3.3 Syme

* 第二关《祸水东引》的玩家角色。
* 当前身份：被边缘化的 Argon 战略档案官与临时防务顾问。
* 能力：

  * 擅长 OOS 多点调度；
  * 熟悉 HAT 舰队；
  * 熟悉 Yaki 导航信标；
  * 能够快速理解 Xenon 路径异常；
  * 能利用高速公路组织小型舰艇增援。
* 初始处境：

  * 警告远征风险但未被采纳；
  * 没有正式舰队指挥权；
  * Argon 之都遭袭后被迫临时接管防务。
* 本关作用：

  * 作为玩家的战术对手；
  * 通过公开广播组织 HAT、ANT、警察与平民船只抵抗；
  * 迫使玩家看见“敌方单位”背后具体的人；
  * 在最终决战中带领首都残部与回援舰队反击。
* 推荐结局：不死亡。

  * 指挥舰被玩家击毁或瘫痪；
  * HAT 舰船秘密救走其逃生舱；
  * 为后续由 ANT 组织联合反攻保留核心人物。

### 3.4 Argon 远征舰队司令

* 负责占领金氏家族与原 ZYA 领地。
* 相信这次行动将成为 Argon 扩张的历史转折。
* 开始时拒绝因首都出现“小规模异常”取消远征。
* 后来被 Xenon 抄断后路，被迫抛弃航母、补给舰、工程船和伤员，让驱逐舰群单独突围。
* 其通讯用于表现 Argon 主力从自信到混乱、再到绝望回援的变化。

### 3.5 HAT 临时增援队

* 由 Syme 通过旧关系召集。
* 并非第 86 护卫队本身；86th 已向 Boron 方向撤离。
* 组成：地狱犬、弥诺陶洛斯、战机与少量武装运输船。
* 叙事作用：

  * Argon 曾把 HAT 当作前线耗材；
  * Argon 首都燃烧时，最先回来保护平民的仍然是 HAT。

### 3.6 ANT 增援

* 通过高速公路和星门网络不断进入 Argon 之都。
* 主要为 S/M 舰，主力舰无法及时通过高速环带赶到。
* 本关结尾可以由 ANT 舰船救走部分 Argon 政府、平民与 Syme。

## 4. 玩家单位与伪 RTS 系统

### 4.1 第一阶段座驾：Xenon F

定位：高速侦察、渗透和信标部署平台。

建议特性：

* 速度高；
* 武装弱或无武装；
* 具有低可探测模式；
* 可短时间关闭高功率系统，降低扫描概率；
* 携带唯一战略锚点信标；
* 被完全扫描或摧毁则任务失败。

玩家在此阶段不应主动战斗。

### 4.2 第二阶段旗舰：Xenon CPU 母舰

定位：

* 第二舰队在星门网络内的临时生产、指挥和跃迁节点；
* 玩家意识与监管权限的主要载体；
* 伪 RTS 系统核心；
* 本关必须保护的关键单位。

失败条件：CPU 母舰被摧毁。

### 4.3 制造配额

游戏内部可以使用数字资源实现，但剧情和 UI 不称为“钱”，统一称为：

> 制造配额

获取方式：

* CPU 母舰每隔固定时间生成；
* 摧毁 Argon 工业和大型舰船获得额外回收配额；
* 占领或摧毁特定资源仓库获得一次性配额；
* 完成可选目标获得上级额外授权。

推荐单位费用仅作为初始平衡参考：

| 单位                    |  推荐费用 | 定位                   |
| ----------------------- | --------: | ---------------------- |
| Xenon M 小队            |       100 | 高速截击、高速公路战斗 |
| Xenon PE                |       250 | 护卫与防空             |
| Xenon H                 |       600 | 反主力与重型支援       |
| Xenon K                 |      1000 | 造船厂与空间站攻坚     |
| Xenon I                 |      2200 | 战略压制               |
| 门禁制压站核心          |      2000 | 第三部分必须部署       |
| 门禁制压站炮塔/护盾升级 | 500—1000 | 提升堵门能力           |

实际实现时可以只提供有限按钮，不必接入完整经济链。

### 4.4 推荐生产选择

玩家需要在三种需求之间权衡：

* 生产大量 M/PE，应对高速公路涌入的 ARG、ANT、HAT 战机；
* 储存配额制造 K/I，加速摧毁造船厂和空间站；
* 提前投入门禁制压站，为第三部分回援舰队做好准备。

如果玩家把配额全部用于进攻，第三部分堵门阵地会较弱。

如果玩家过早投资防御，第二部分摧毁 Argon 造舰体系的速度会变慢。

## 5. 星区、布景与初始状态

### 5.1 Argon 之都

用途：

* 第二阶段主要战场；
* Argon 船坞、造船厂、建设协调中心所在地；
* 高速公路增援的核心区域；
* 第三阶段堵门决战所在地。

初始布置：

* Argon 大型造船厂；
* Argon S/M 船坞；
* Argon 联邦建设协调中心；
* 3—5 条可移动建造船；
* 约 10 条巨兽驱逐舰；
* 2—3 条巨像航母；
* 大量警察、战机和商船；
* 多条高速公路连接点；
* 通往希望之歌的选择与希望之歌的信仰的星门。

Argon 之都不应显得毫无防备，而应显得：

> 防御力量足以应对传统星门入侵，却不足以应对从死胡同方向直接出现的 CPU 母舰。

### 5.2 希望之歌的选择

用途：

* 第一阶段潜入起点；
* Argon 远征舰队出征检阅；
* 远征舰队进入塔卡与原 ZYA 领地的通道；
* 第三部分回援舰队返回 Argon 之都的方向。

布景：

* 大量巨兽、巨像、补给舰、工程船和舰载机组成出征队列；
* 公开广播、新闻无人机和民用观礼船；
* 阅兵结束后，大部分主力逐批离开。

为控制性能，可只渲染部分核心舰队，其余以远景质量信号、通讯和预录镜头表现。

注：希望之歌的选择与Argon之都的星门断开，防止玩家在后续阶段误入。

### 5.3 希望之歌的信仰

用途：

* Xenon F 潜入路线；
* Argon 低优先级巡逻区；
* 通往诺皮利奥纪念碑的死胡同航道。

布景：

* 少量警察巡逻；
* 商船和旅游船；
* 仅设置HAT的巡逻小队；

### 5.4 诺皮利奥纪念碑

用途：

* 第一枚战略信标部署点；
* CPU 母舰跃迁入口；
* 历史演出节点。

布景：

* M0 旧战场残骸；
* 纪念信标与布伦南相关广播；
* 微弱空间异常；
* 远离常规航道。

### 5.5 其他星区

主要用于布景，给“星门陨落”的场景做背景板

* 王国末路
* 三圣殿
* 大交易所

## 7. 详细 Phase 剧本

## Phase 0：监管实例上线

### 目的

建立玩家身份、Xenon 视角和第二舰队的冷静语言风格。

### 场景

* 黑屏中先出现 CPU 检查文本；
* 玩家意识接入 Xenon F；

## Phase 10：堵门决战

### 敌军组成

建议分 3—4 波穿门，避免瞬间拥堵。

第一波：

* 5—6 条受损巨兽；
* 少量护卫舰；
* 先遣无人机。

第二波：

* 6—8 条巨兽；
* ANT 鱼雷机；
* HAT 护卫舰。

第三波：

* 远征旗舰与剩余驱逐舰；
* Syme 的临时指挥舰；
* 少量从首都撤回的警察与民用武装船。

### Syme 战术

* 第一波正面吸收空间站火力；
* 第二波鱼雷机绕击站点护盾模块；
* HAT 护卫舰攻击 CPU 母舰或站点建设核心；
* 第三波集中突破一侧火力死角；
* Syme 不追求击败全部 Xenon，只尝试为幸存舰船打开通道。

### 公开广播

* 624【玩家】敌方请求以关闭军事节点换取舰队通行。
* 625【第二舰队指挥核心】拒绝。幸存舰队具备重建与传播 SSD 型技术的工业价值。
* 626【玩家】其当前任务是撤离，不是重建。

歼灭所有穿门的 Argon 主力舰，并摧毁 Syme 指挥舰。

## 9. 评分设计建议

### 9.1 推荐评分维度

| 评分项                 | 说明                               |
| ---------------------- | ---------------------------------- |
| 潜入隐蔽度             | Xenon F 暴露次数、是否触发高级警戒 |
| 信标部署效率           | 抵达纪念碑与校准耗时               |
| CPU 母舰结构           | 全关核心单位保护情况               |
| 制造配额效率           | 是否大量浪费、是否完成回收目标     |
| 船坞摧毁时间           | 小型舰生产压制效率                 |
| 造船厂摧毁时间         | 主力生产压制效率                   |
| 建造船拦截数           | 是否阻止 Argon 快速重建            |
| 门禁制压站完整度       | 最终堵门准备情况                   |
| Xenon 主力舰损失       | 玩家战略调度表现                   |
| Argon 回援舰队歼灭效率 | 第三部分表现                       |
| 可选目标               | 捕获情报、摧毁备用建设节点等       |

### 9.2 推荐失败条件

* Xenon F 在信标部署前被摧毁或完整扫描；
* 战略信标被摧毁；
* CPU 母舰被摧毁；
* 未在时间限制内摧毁 Argon 建设协调中心，导致建造船逃离数量超过上限；
* 门禁制压站在回援舰队抵达前被摧毁；
* Argon 回援主力突破星门防线并抵达 CPU 母舰；
* 玩家关键监管连接中断。

### 9.3 星级设计原则

* 一星应允许玩家以较大损失完成主线，不要求完美摧毁所有敌舰；
* 五星应奖励正确利用制造配额和提前建设堵门站；
* 不应鼓励玩家无限拖延刷击毁数；
* 所有循环生成敌人的得分应设上限；
* 潜入阶段被轻微发现不应直接导致失败，而应影响评分和后续敌军强度。

## 10. 关键实现约束

1. Xenon F 潜入应使用多级暴露度，不建议一次扫描直接失败。
2. Argon 远征舰队必须通过检阅和离场镜头明确展示，不能只用对白说明。
3. Argon 之都不能表现为完全无防御，而应体现“防御方向错误”。
4. 诺皮利奥纪念碑需要明确的空间残响与 M0 历史意义。
5. CPU 母舰生产资源统一称为“制造配额”，不要在剧情中称为 credits。
6. 第二部分必须同时存在战机潮、主力舰攻坚与建造船逃离，体现伪 RTS 调度。
7. 高速公路主要输送 S/M 增援，回援主力舰必须从星门进入。
8. 门禁制压站需要玩家提前投资，不能在第三部分凭空出现完整防线。
9. Argon 远征舰队回援前必须播放“抛弃航母、补给舰和工程船”的通讯。
10. Syme 应根据玩家生产和进攻方向进行有限动态反制。
11. Syme 指挥舰可以被判定摧毁，但角色本人应通过隐藏演出保留生还可能。
12. 玩家与 Argon 的通讯保持单向：玩家可以截听，但不能直接回复。
13. 多星区同步演出应短而有差异，避免连续播放相同的“传送—爆炸—求救”。
14. TER 必须成为唯一成功阻止稳定信标的大型文明，解释后续太阳系仍能组织抵抗。
15. 最终口号前必须保留明显停顿，不能让玩家立即悔悟或公开叛变。

## 11. 本关人物弧光总结

玩家的变化不是从“坏人”变成“好人”，而是从确定无疑变成开始看见矛盾。

开场时，他认为自己是历史受害者的继承人。

在诺皮利奥纪念碑，他陶醉于成为历史审判者。

在摧毁 Argon 工业时，他通过制度语言把具体生命重新定义为可清除目标。

Syme 让他第一次发现：

* 同样遭到制度背叛的人，可以选择保护别人，而不是审判所有人；
* HAT 曾被 Argon 牺牲，却仍然回来保护 Argon 平民；
* Argon 回援舰队并非只为政府，而是为了“回家”。

玩家最终仍然摧毁了他们。

当整个星门网络同步陷落时，玩家发现第二舰队正在复制地球当年的逻辑：

> 因为对方未来可能成为威胁，所以现在就应当被全部销毁。

他提出疑问，上级把疑问定义为认知偏差，并提醒他“连续性不会中断”。

玩家已经开始怀疑：

* 自己究竟是不是人类；
* 自己是否只是可以替换的监管实例；
* 第二舰队是否仍在为生存而战；
* 或者早已把生存变成了统治和灭绝的借口。

但他没有在本关反抗。

他看见之后，仍然重复了口号。

## 12. 本关与后续战役的关系

《审判日》结束后：

* Argon 之都造舰与建设体系被毁；
* Argon 远征主力覆灭；
* Argon 政府失去统一调度能力；
* ANT 将逐渐接管人类阵营的组织责任；
* Syme 可能被 HAT 救出，并成为后续联合反攻的重要情报与指挥人物；
* Boron、TEL、TRI、ZYA 腹地同时出现 Xenon；
