# Gatefall Chronicles

一句话简介：这是一个基于《X4: Timelines》的原创隐藏时间线战役，讲述 Xenon 借助异常跃迁技术撕开未来战局，各阵营在灭世危机前被迫重新结盟的故事。

当前公开版本包含三关可完整游玩并结算的战役：

- 《星门陨落01：暗流涌动》
- 《星门陨落02：祸水东引》
- 《星门陨落04：同舟共济》

English version: [README_EN.md](README_EN.md)

## 最新更新：《同舟共济》

本次更新加入了完整的第四关《星门陨落04：同舟共济》：

- 驾驶 Boron 九头蛇御用型，保护打捞船回收 Xenon 远程炮击兵器情报。
- 与 Boron–Terran 联合舰队突入异端的终结，摧毁两艘 Xenon 雷达船及其护卫力量。
- 掩护移动基地打包船，并获得一座归玩家控制的战术生产平台。
- 使用 S/M 与 L 造船厂、定制蓝图、海豚补给机制和异构石墨炔资源循环建立舰队。
- 以独立支线目标摧毁 Xenon 造船厂与船坞，削弱持续刷新的敌军增援。
- 摧毁两座 M0 护盾能源供给点，并最终围攻 Xenon 远程炮击舰 M0。
- 加入多段分镜、动态 BGM、失败条件、详细计分和一至五星评级。

同时修复了多项第四关流程问题，包括 Xenon 舰队停火、增援只刷新一轮、任务目标无法推进、关键单位无敌状态无法解除，以及结算细则不显示等问题。

## 安装方法

如果你是从 GitHub 获取源码，安装方式如下：

1. 下载或克隆本仓库。
2. 将 `extensions/gatefall_chronicles` 整个文件夹复制到 X4 游戏根目录下的 `extensions` 文件夹中。
3. 启动游戏，在扩展菜单中确认 `Gatefall Chronicles` 已启用。

典型目录结构：

```text
X4 Foundations/
└─ extensions/
   └─ gatefall_chronicles/
```

## 进入游戏的流程

本战役不是普通主宇宙任务，而是通过隐藏时间线战役入口进入。

1. 启动游戏并进入 `时间线（Timelines）`。
2. 进入时间线 Hub，也就是玩家常说的“黑色空间”。
3. 找到 `隐藏战役 / 隐藏场景` 入口。
4. 打开隐藏战役列表。
5. 在列表末尾找到 `Gatefall Chronicles` 的战役入口。
6. 选择希望游玩的关卡。

如果你已经游玩过官方隐藏场景，进入方式基本一致；本 Mod 的战役挂在同一个隐藏战役列表中。

## 当前公开内容

### 战役 01：暗流涌动

当前状态：公开 Demo 已完成，可完整游玩与结算

- 玩家与 TER 盟友僚机前往阿尔特弥斯的朦胧部署远程侦察卫星。
- 调查过程中发现 Kha'ak 巢穴已被摧毁，并出现异常空间折叠信号。
- 玩家被卷入 Xenon 核心区野蛮疾驰，随后突围至 TER 前哨站。
- 坚持到 Terran 干预军战列舰 `TER 典范` 抵达并接管战场。

主要特点：护卫舰战斗、主力舰引擎破坏、前哨站防守、多阶段分镜与评分结算。

### 战役 02：祸水东引

当前状态：完整战役 Demo 已完成，可完整游玩与结算

- 玩家扮演 Argon 后方指挥官，在希望之歌的选择远程指挥 HAT 第 86 护卫队。
- 通过地图进行 OOS 舰队指挥，处理货船、矿船和空间站遭遇的多点 Xenon 袭击。
- 潜入机密数据库，取得能够改变 Xenon 攻击向量的 Yaki 导航协议。
- 在 Argon 决定牺牲第 86 护卫队后，将 Xenon 主力引向 ZYA 的金氏家族，为护卫队争取逃生机会。

主要特点：站内流程、OOS 远程指挥、多线防守、数据库潜入、导航信标部署与评分结算。

### 战役 04：同舟共济

当前状态：完整战役已完成，可完整游玩与结算

- 保护打捞船从虹鳉航母残骸中取得异端的终结封锁情报。
- 率领 Boron–Terran 联合舰队摧毁 Xenon 雷达船，解除远程炮击威胁。
- 部署玩家控制的战术生产平台，并通过造船、资源补充和舰队编组推进战线。
- 对 Xenon 供能设施、造船厂、船坞和 M0 发动一场接近 RTS 玩法的完整攻势。

主要特点：大规模舰队战、玩家造船厂、动态资源机制、独立支线目标、循环敌军增援、定制 BGM、完整分镜、失败判定与五星评分。

说明：当前公开内容以中英文字幕演出为主，暂无语音。
