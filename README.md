# Gatefall Chronicles

一句话简介：这是一个基于《X4: Timelines》的原创隐藏时间线战役，讲述 Xenon 利用异常跃迁技术撕开未来战局、各阵营被迫在灭世危机前重新结盟的故事。

当前公开版本为《暗流涌动》第一阶段 Demo，已经可以完整游玩、结算并体验战役开篇。

English version: [README_EN.md](README_EN.md)

## 依赖

本 Mod 目前依赖：

* X4: Foundations
* X4: Timelines
* X4: Cradle of Humanity

## 安装方法

如果你是从 GitHub 获取源码，安装方式如下：

1. 下载或克隆本仓库。
2. 将 `extensions/gatefall_chronicles` 整个文件夹复制到你的 X4 游戏根目录下的 `extensions` 文件夹中。
3. 启动游戏，在扩展菜单中确认 `Gatefall Chronicles` 已启用。

典型目录结构示例：

```text
X4 Foundations/
└─ extensions/
   └─ gatefall_chronicles/
```

## 进入游戏的流程

本战役不是普通主宇宙任务，而是一个隐藏时间线战役入口。

进入流程如下：

1. 启动游戏，进入 `时间线（Timelines）`。
2. 进入时间线 Hub，也就是玩家常说的“黑色空间”。
3. 在黑色空间中找到 `隐藏战役 / 隐藏场景` 的入口。
4. 打开隐藏战役列表。
5. 在隐藏战役列表的最后，找到我们的战役入口。
6. 选择 `Gatefall Chronicles / 暗流涌动` 开始游玩。

如果你已经游玩过官方隐藏场景，流程基本一致；我们的战役是挂在同一个隐藏战役列表中的。

## 当前公开内容

### 战役 01：暗流涌动

当前状态：已完成公开 Demo，可游玩

内容概要：

* 玩家与 TER 盟友僚机前往阿尔特弥斯的朦胧部署高级人造卫星。
* 调查过程中发现 Kha'ak 巢穴已被摧毁，并出现异常信号。
* 玩家被异常空间折叠卷入 Xenon 核心区野蛮疾驰。
* 在逃离与前哨站防守后，TER 干预军 Asgard `TER 典范` 抵达接管战场。

当前版本特点：

* 完整字幕
* 多段任务目标切换
* 开场、异常、跃迁、增援等分镜演出
* 可结算评分与星级

说明：

* 当前公开的是第一关 Demo
* 暂无语音，使用字幕演出
