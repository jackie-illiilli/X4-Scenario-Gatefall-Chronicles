# Gatefall Chronicles

一句话简介：这是一个基于《X4: Timelines》的原创隐藏时间线战役，讲述 Xenon 借助异常跃迁技术撕开未来战局，各阵营在灭世危机前被迫重新结盟的故事。

当前公开版本已包含前两关可游玩的 Demo：

- 《星门陨落01：暗流涌动》
- 《星门陨落02：祸水东引》

English version: [README_EN.md](README_EN.md)

## 依赖

本 Mod 当前依赖：

- X4: Foundations
- X4: Timelines
- X4: Cradle of Humanity
- X4: Split Vendetta

## 安装方法

如果你是从 GitHub 获取源码，安装方式如下：

1. 下载或克隆本仓库。
2. 将 `extensions/gatefall_chronicles` 整个文件夹复制到你的 X4 游戏根目录下的 `extensions` 文件夹中。
3. 启动游戏，在扩展菜单中确认 `Gatefall Chronicles` 已启用。

典型目录结构示例：

```text
X4 Foundations/
└─ extensions/
   └─ gatefall_chronicles/
```

## 进入游戏的流程

本战役不是普通主宇宙任务，而是一个隐藏时间线战役入口。

进入流程如下：

1. 启动游戏，进入 `时间线（Timelines）`。
2. 进入时间线 Hub，也就是玩家常说的“黑色空间”。
3. 在黑色空间中找到 `隐藏战役 / 隐藏场景` 的入口。
4. 打开隐藏战役列表。
5. 在隐藏战役列表的最后，找到我们的战役入口。
6. 选择 `Gatefall Chronicles / 星门陨落01：暗流涌动` 开始游玩。

如果你已经游玩过官方隐藏场景，流程基本一致；我们的战役挂在同一个隐藏战役列表中。

## 当前公开内容

### 战役 01：星门陨落01：暗流涌动

当前状态：已完成公开 Demo，可完整游玩与结算

内容概要：

- 玩家与 TER 盟友僚机前往阿尔特弥斯的朦胧部署远程侦察卫星。
- 调查过程中发现 Kha'ak 巢穴已被摧毁，并出现异常信号。
- 玩家被异常空间折叠卷入 Xenon 核心区野蛮疾驰。
- 在逃离与前哨站防守后，TER 干预军 Asgard `TER 典范` 抵达接管战场。

当前版本特点：

- 完整字幕
- 多阶段任务目标切换
- 异常信号、跃迁、增援等分镜演出
- 可结算评分与星级

### 战役 02：星门陨落02：祸水东引

当前状态：已完成公开 Demo，可完整游玩与结算

内容概要：

- 玩家扮演 Argon 后方指挥官，在希望之歌的选择远程指挥 HAT 第 86 护卫队。
- 玩家需要连续处理 TEL 货船、HAT 矿船与 HAT 空间站遭遇的多点 Xenon 袭击。
- 在确认 Xenon 主力正在塔卡的瀑布深处集结后，玩家命令队长机部署人造卫星并死守空间站。
- 当 Argon 决定牺牲第 86 护卫队时，玩家盗用 Yaki 导航协议，将 Xenon 主力引向 ZYA 的金氏家族，为护卫队争取逃生机会。

当前版本特点：

- 站内出生与动态室内流程
- 通过地图进行 OOS 远程指挥
- 多阶段防守、侦察、潜入数据库与部署导航信标流程
- 完整字幕、BGM 切换、结算评分与星级

说明：

- 当前公开内容仍然以字幕演出为主，暂无语音。
- 前两关已经可以独立完整游玩并结算。
